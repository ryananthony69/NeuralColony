# AGENTS.md

## Project purpose
This repo builds a RimWorld-style AI colony demonstration game called NeuralColony.

## Hard constraints
- $0 budget
- no API keys
- no cloud dependency
- no required online services
- fully local simulation-first implementation

## Priority order
1. Demonstration clarity
2. Readability
3. Stable local Windows execution
4. Modular architecture
5. Optional local extensibility later

## Current milestone focus
Milestone 1 only.

## Milestone 1 done criteria
- top-down camera
- node actors / colony members
- family tree support
- visible conversations
- visible work / project construction
- persistent inspector
- goal input panels
- time controls
- working local Windows run instructions
- no internet required
- no API keys required

## Rules
- Do not jump ahead to backend integration
- Do not optimize for photorealism
- Do not over-engineer networking
- Prioritize a simulation that teaches the system concept clearly

## Repo guidance
- Docs are in Docs/
- Unreal assets go in Content/
- C++ goes in Source/
