NeuralColony Content Scaffolding (Milestone 1)

This folder tree is a non-breaking scaffold for future free asset integration.

Rules:
1) Put raw downloaded packages into /Game/NeuralColony/ArtSources first.
2) Import cleaned/game-ready assets into Meshes/Materials/Textures/etc.
3) Keep naming stable (SM_NC_*, SK_NC_*, MI_NC_*).
4) Update Docs/AssetReplacementManifest.csv whenever replacing a placeholder.
5) Do not add runtime internet downloading.
6) Preserve local/offline simulation behavior.
