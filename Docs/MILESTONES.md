# MILESTONES

## Milestone 1 - Demonstration Layer Vertical Slice
Build a playable top-down prototype with:
- top-down camera with pan and zoom
- node actors / NPCs
- node states: baby, idle, running, complete, error
- family tree data support
- visible information passing via speech / thought bubbles
- visible project build sites with completion progress
- left panel: My Goal / Neural Network Goal
- right panel: persistent inspector
- top status bar
- alert / event feed
- time controls: pause, 1x, 2x, 3x
- mock / simulated colony runtime only
- no internet requirement
- no API requirement

### Acceptance criteria
- launches locally on Windows
- camera works
- selecting a node updates inspector
- goal text is editable
- time controls affect simulation speed
- nodes visibly change state
- conversations and construction progress are visible
- no real backend required
- no API keys required

## Milestone 2 - Colony Systems
Add:
- specialization roles

- richer job priorities
- parent / child lifecycle growth
- project stage pipeline
- better alerts

## Milestone 3 - Optional Local Backend Attachment
Optional local-only attachment:

- local process / file integrations
- local-only tooling

## Milestone 4 - Full Closed Loop
Add:
- verifier role
- patch planner role
- runner role
- project-level pass/fail criteria
- visible repair loop
