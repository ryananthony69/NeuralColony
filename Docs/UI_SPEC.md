# UI SPEC

## Core layout
Three-column management layout over a top-down world.

### Left panel
- My Goal
- Neural Network Goal
- project summary
- priorities
- time controls

### Center
Top-down colony view with:
- architect core
- specialist nodes / colony members
- build sites
- room / zone layout
- speech bubbles
- progress bars
- optional connection overlays

### Right panel
Persistent inspector for selected node:
- role
- status
- activity
- specialization
- parents / generation
- linked project
- current task
- load / health
- recent outputs

### Top bar
- node count
- active jobs
- failures
- colony health
- current phase
- active projects

### Notifications
RimWorld-style alert feed:
- new node born
- specialization discovered
- verification failed
- repair applied
- project completed

### Time controls
- pause
- 1x
- 2x
- 3x

### Node states
- baby
- idle
- queued
- running
- complete
- retrying
- error
